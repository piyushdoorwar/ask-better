Privacy Policy & Data Processing

AskBetter is local-first. This summary explains what is stored, what is sent, and what is not sent.

Read full Privacy Policy

What Is Stored Locally

  • Settings: provider, model, preset, integration toggles, and UI preferences.
  • Keys: your Gemini, OpenAI, and/or Anthropic API keys are saved in chrome.storage.local.

What Is Sent Externally

  • Optimize request: your current prompt is sent only to the selected provider API, Google Gemini, OpenAI, or Anthropic.
  • Request metadata: selected provider model and preset instruction are sent with the request.
  • Custom Prompt Additions: if set, they are sent as additional guidance with optimize requests.
  • Key test: a direct provider key validation call is made when you click Test key.

What Is Not Sent

  • No AskBetter backend server receives your prompts or optimized outputs.
  • No AskBetter analytics or telemetry are collected in this version.
  • API keys are not injected into webpage DOM by content scripts.

Privacy Policy

AskBetter privacy practices for the entire extension are documented in the full policy: https://piyushdoorwar.github.io/policy/ By installing, enabling, or using AskBetter, you acknowledge this policy.
