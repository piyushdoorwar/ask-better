Privacy Policy & Data Processing

PromptForge is local-first. This summary explains what is stored, what is sent, and what is not sent.

What Is Stored Locally

  • Settings: provider, model, preset, integration toggles, and UI preferences.
  • Keys: provider API keys are saved in chrome.storage.local.
  • Optional stats: only usage counters when analytics opt-in is enabled.

What Is Sent Externally

  • Optimize request: your current prompt is sent only to the selected provider API.
  • Provider metadata: selected model and preset instruction are sent with the request.
  • Key test: a direct provider key validation call is made when you click Test key.

What Is Not Sent

  • No PromptForge backend server receives your prompts or optimized outputs.
  • No prompt text is included in analytics counters.
  • API keys are not injected into webpage DOM by content scripts.
